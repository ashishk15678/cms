My External Portfolio

This is a completely separate, static HTML page demonstrating how easy it is to embed the CMS on any external site using the script tag.

How to use on your own site:

1. Add a target div where you want the posts to appear:

<div id="cms-embed"></div>

2. Add the script tag right before your closing </body> tag:

<script
  src="http://localhost:3000/embed.js"
  data-target="cms-embed"
  data-mode="list"
></script>
